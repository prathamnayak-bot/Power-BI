# Student AI Impact Dashboard

## 📊 Overview

This project analyzes how Generative AI usage impacts student academic performance, study habits, and mental well-being.

## 🛠 Tools Used

* Power BI
* DAX
* Data Cleaning (Power Query)

## 📈 Key Insights

* Moderate AI usage improves GPA performance.
* High AI dependency reduces skill retention.
* Students with higher AI usage show increased burnout risk.
* Traditional study hours still play a crucial role in performance.

## 💡 Recommendations

* Encourage balanced use of AI tools.
* Promote traditional study methods alongside AI usage.
* Monitor burnout levels among high AI users.
* Train students in effective prompt engineering for better outcomes.
